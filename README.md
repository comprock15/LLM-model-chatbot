# LLM-model-chatbot
Данный репозиторий содержит код для запуска телеграм-бота, использущего LLM, запущеной на этом компьютере. Бот умеет сохранять контекст.

| Команда | Описание |
| ------- | -------- |
| /start | Начало работы и список команд |
| /model | Используемая модель |
| /clear | Очистка контекста |
